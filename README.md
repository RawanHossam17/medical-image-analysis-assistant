# Medical Image Analysis Assistant

This project applies computer vision techniques on medical images (HAM10000 dataset) for analysis and classification.

## Dataset

HAM10000 Skin Cancer Dataset
https://www.kaggle.com/datasets/kmader/skin-cancer-mnist-ham10000

## Pipeline

* Preprocessing (Filtering)
* Harris Corner Detection
* Pyramid Analysis
* SIFT Matching
* Segmentation
* Classification


## Structure

* notebooks/  experiments
* results/  outputs
* report/  final report
